========================================================================
    スタティック ライブラリ: Strmbase プロジェクトの概要
========================================================================

簡易キーチェンジャーをビルドする際に必要となるライブラリです。

このライブラリをビルドするには、「Microsoft Windows SDK for Windows 7 and .NET Framework 4」
https://www.microsoft.com/en-us/download/details.aspx?id=8279
をインストールすると配置される、
Samples\multimedia\directshow\baseclasses
フォルダーの中にあるすべての .cpp / .h ファイル（amextra.cpp など）を、ソリューションファイルと同じフォルダーに
保存してから、ソリューションファイルを開き、ビルドして下さい。
